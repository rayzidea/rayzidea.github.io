Jupyter Notebook 이란?

웹 브라우저에서 코드를 직접 작성하고 실행결과까지 볼 수 있는 오픈소스 웹 앱이다. 나 같은 개.알.못도 쉽게 다가갈 수 있는 다재다능한 문서편집기라고나 할까? 자세한 내용은 공식 웹사이트에서 확인해보고, Jupyter Notebook을 더 편하게 쓸 수 있는 단축키들을 알아보자.

Jupyter Notebook 단축키

Command Mode에서의 단축키

1) Edit Mode에서 Command Mode 들어가기: Esc
2) Markdown 셀로 전환: M
3) Code 셀로 전환: Y
4) 위 행에 셀 삽입: a
5) 아래 행에 셀 삽입: b
6) Command Mode에서 Edit Mode로 빠져나오기: Enter

(공부하면서 배운 것들로 계속 업데이트 됩니다)
